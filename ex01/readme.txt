cription:
--------------------
This program checks if the digit '3' exists in a given number. It iterates through each digit of the number and outputs "Yes" if '3' is found, otherwise it outputs "No".

Test Cases:
-----------
1. Input: 12345
   Output: Yes
   Explanation: The digit '3' is present in the number 12345.

2. Input: 98765421
   Output: No
   Explanation: The digit '3' is not present in the number 987654321.

3. Input: 333
   Output: Yes
   Explanation: All digits of the number 333 are '3', hence "Yes" is printed.
